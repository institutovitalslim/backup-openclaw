# systeme.md - Systeme.io

- Cursos hospedados em *.systeme.io exigem sessao HTTP autenticada ou navegador funcional.
- Para checagem deterministica, preferir um script nao interativo que valide login, matricula e primeira aula.
- Para navegacao com browser, abrir /school, localizar a matricula e depois navegar para /school/course/<course_path>.
- Se o login autenticar mas a matricula nao aparecer, reportar falta de inscricao ou conta incorreta.
- Se o item do 1Password tiver URL mas campos inconsistentes, parar e reportar conflito de credenciais.
