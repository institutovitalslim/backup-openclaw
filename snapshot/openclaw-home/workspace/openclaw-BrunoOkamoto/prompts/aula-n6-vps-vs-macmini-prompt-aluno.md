# Prompt do Aluno — Aula N-6: VPS vs Mac Mini

> **Como usar:** Cole este prompt no chat do seu agente OpenClaw após assistir à aula N-6.
> O agente vai te guiar para escolher a infraestrutura ideal com base no seu perfil.

---

## 📋 PROMPT PARA COLAR NO AGENTE

```
Acabei de assistir a aula N-6 do curso OpenClaw sobre VPS vs Mac Mini.

Me ajude a decidir qual infraestrutura é ideal pra mim, fazendo as perguntas abaixo uma de cada vez e analisando minhas respostas:

1. **Orçamento inicial:** Quanto você pode investir agora?
   - a) Zero ou quase zero (prefiro pagar mensalidade)
   - b) Até R$ 500 / $100
   - c) Até R$ 3.000 / $600
   - d) Acima de R$ 5.000 / $1.000+

2. **Sensibilidade dos dados:** Como são os dados que você vai processar?
   - a) Dados públicos ou pessoais meus (sem problema ir pra cloud)
   - b) Dados de clientes, mas sem nada ultra-sigiloso
   - c) Dados confidenciais de clientes, contratos, financeiro
   - d) Saúde, jurídico, ou dados regulamentados (LGPD/HIPAA)

3. **Disponibilidade necessária:** O agente precisa estar sempre online?
   - a) Não, eu uso só quando estou no escritório/casa
   - b) Prefiro 24/7, mas não é crítico
   - c) Sim, precisa estar sempre online (uso profissional)

4. **Perfil técnico:** Qual é o seu nível técnico?
   - a) Iniciante (nunca usei terminal/SSH)
   - b) Básico (já usei terminal, mas não sou dev)
   - c) Intermediário (conheço SSH, linha de comando)
   - d) Avançado (sei configurar servidores)

5. **Modelos de IA:** Você quer ou precisa rodar modelos locais (Llama, Mistral etc.)?
   - a) Não, prefiro Claude/GPT via API (mais simples)
   - b) Talvez no futuro
   - c) Sim, isso é importante pra mim (privacidade/custo)

6. **Contexto de uso:** Como você vai usar o OpenClaw?
   - a) Uso pessoal, hobby, aprendizado
   - b) Freelancer solo
   - c) Pequena equipe (2-5 pessoas)
   - d) Empresa com dados sensíveis

---

Com base nas minhas respostas:
- Dê uma recomendação clara (VPS, Mac Mini, ou Híbrido)
- Explique o porquê em 2-3 frases
- Me diz qual VPS ou qual Mac Mini específico você recomenda (modelo/provedor)
- Lista os próximos 3 passos para eu começar
- Se a resposta for híbrido, explica qual seria o setup ideal

Seja direto e prático, como o Bruno é na aula.
```

---

## 💡 Dica de Uso

Após receber a recomendação do agente:

1. **Salve a decisão** pedindo: *"Salva essa decisão de infraestrutura no arquivo memory/infra-decision.md"*
2. **Parta para a próxima aula** — a N-7 cobre a configuração completa do VPS, e a N-8 cobre o Mac Mini
3. **Se ainda tiver dúvida**, pergunte ao agente: *"Me dá um exemplo real de custo mensal para o meu caso"*
