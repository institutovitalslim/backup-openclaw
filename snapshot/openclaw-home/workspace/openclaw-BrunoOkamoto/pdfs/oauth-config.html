⚠️ Leia Antes de Começar

🚨 A Anthropic bloqueou OAuth para planos Pro

Se você tem assinatura Claude Pro ($20/mês), o fluxo setup-token / openclaw setup-token NÃO FUNCIONA MAIS para você. Isso afeta a maioria dos alunos e é a dúvida #1 do curso (~20 ocorrências nos fóruns).

Sintoma: Você tenta rodar o fluxo OAuth, parece funcionar, mas na primeira mensagem aparece erro "OAuth flow not supported for Pro plans" ou "Unauthorized".

Solução: Ignorar completamente o fluxo OAuth e usar uma API Key direta — que é o que vamos aprender aqui.

02
Por Que o OAuth Quebrou

Em 2026, a Anthropic decidiu que o fluxo OAuth só funciona para planos API (pay-per-use). Para planos Pro ($20/mês) — que é o que a maioria tem — eles bloquearam. Separação clara: "assinatura para usar no chat" vs "acesso programático via API".

Situação Funciona? O que fazer
Plano Pro ($20/mês) + OAuth ❌ Bloqueado Criar API Key separada
Plano gratuito + OAuth ❌ Bloqueado Criar API Key com billing
API Key direta (qualquer plano) ✅ Funciona O que vamos fazer
💳 Tenho Claude Pro — preciso pagar a mais?
Sim, infelizmente. Sua assinatura Pro dá acesso ao claude.ai (interface web), não à API. Para o OpenClaw, você precisa de billing separado no console.anthropic.com. A boa notícia: com modelos econômicos (Haiku para tasks simples), o gasto real costuma ser $5-15/mês — menos que o Pro.
💡
Dica: migrar do Pro para API-only

Você pode cancelar o Pro e usar só a API. Perde o acesso ao claude.ai (a interface web), mas o OpenClaw substitui isso com muito mais poder.

03
Gerando API Keys
1
🤖 API Key — Anthropic
~5 min
  1. Acesse console.anthropic.com — atenção: é o console, não o claude.ai
  2. Faça login ou crie uma conta
  3. No menu lateral: Settings → API Keys
  4. Clique em "Create Key"
  5. Dê um nome descritivo: openclaw-pessoal
  6. Copie a chave — começa com sk-ant-api03-
  7. Verifique billing: Settings → Billing — cartão cadastrado ✅
🚨
COPIE AGORA — Única chance!

A Anthropic só exibe a chave completa UMA VEZ. Após fechar o modal, não é possível visualizá-la novamente. Salve num gerenciador de senhas (1Password, Bitwarden).

⚠️
Sem billing = erro 402 em toda chamada

Verifique se tem cartão cadastrado ou créditos. Sem billing ativo, a chave funciona mas toda chamada retorna HTTP 402 Payment Required.

2
🌐 API Key — OpenAI
~3 min
  1. Acesse platform.openai.com/api-keys
  2. Clique em "Create new secret key"
  3. Dê um nome: openclaw
  4. Copie a chave — começa com sk-proj- ou sk-
  5. Verifique crédito em Billing → Usage
ℹ️
OpenAI é opcional

Configure apenas se pretende usar a OpenAI. Para o curso, recomendamos pelo menos a Anthropic (Claude).

04
Configurar e Verificar
3
⚙️ Configurar com openclaw config
~2 min
openclaw config

Isso abre o assistente interativo. Selecione Anthropic, cole sua API Key quando pedir. Depois selecione OpenAI (se configurado) e cole a chave.

🚫
Se aparecer opção "OAuth" ou "Setup Token" para Anthropic — IGNORE

Escolha sempre "API Key". O wizard ainda mostra essa opção por compatibilidade com contas antigas, mas não funciona mais para a maioria das contas.

4
✅ Verificar com openclaw status
~1 min
openclaw status

Saída esperada quando tudo está correto:

✅ OpenClaw Status ───────────────────────────── Provider: Anthropic (Claude) Model: claude-sonnet-4-5 API Key: sk-ant-...api03 (valid) Status: Connected Provider: OpenAI Model: gpt-4o API Key: sk-proj-...XxXx (valid) Status: Connected ───────────────────────────── All providers configured ✓

05
Erros Comuns
⭐ TOP 1 — "OAuth flow not supported" / "setup-token only"
Causa: Usou fluxo OAuth com plano Pro. Solução: Rode openclaw config de novo, escolha "API Key" (não OAuth), cole sua chave do console.anthropic.com.
"Invalid API Key" / HTTP 401 Unauthorized
Chave errada, expirada ou revogada. Crie uma nova no console e reconfigure com openclaw config.
HTTP 402 — Payment Required
Sem billing ativo. Anthropic: console.anthropic.com → Billing → adicionar cartão. OpenAI: platform.openai.com → Billing → adicionar crédito.
HTTP 429 — Too Many Requests
Rate limit — diferente de credencial inválida! É temporário. Espere 1-5 minutos. Considere upgrade de plano se recorrente.
"API Key not configured"
Nunca rodou openclaw config. Rode agora.
Chave funciona mas agente não responde
tools.profile no modo messaging (v2026.3.2). Execute: openclaw config set tools.profile full
✅ Checklist Final
Conta acessada em console.anthropic.com (não claude.ai)
API Key da Anthropic gerada e salva com segurança
Billing ativo na Anthropic (cartão ou créditos)
(Opcional) API Key da OpenAI gerada e crédito verificado
openclaw config executado com "API Key" (não OAuth)
openclaw status mostrando "Connected" ✅
~/.openclaw/config.json no .gitignore
