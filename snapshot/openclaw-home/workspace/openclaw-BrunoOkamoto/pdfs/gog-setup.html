ℹAutenticação: sempre OAuth 2.0

O gog sempre usou OAuth 2.0 — não existe login por API key simples. Você precisa de um client_secret.json gerado no Google Cloud Console uma única vez. Depois disso, o agente cuida de tudo.

Novidade recente: suporte a ADC (GOG_AUTH_MODE=adc) para Cloud Run / Workload Identity. Para uso pessoal, o fluxo continua sendo OAuth Desktop App.

2⚙️ Setup técnico (uma vez só)

Essa seção é para quem vai configurar o servidor/agente. Se você é usuário final, pule direto pro passo 1 acima.

2.1 Instalar o gog

brew install steipete/tap/gogcli

# verificar
gog --version

2.2 Criar credenciais no Google Cloud Console

  1. Crie um projeto: console.cloud.google.com/projectcreate
  2. Ative as APIs necessárias (links abaixo)
  3. Configure OAuth consent screen (External, Testing): console.cloud.google.com/auth/branding
  4. Adicione sua conta como Test User: console.cloud.google.com/auth/audience
  5. Crie o OAuth Client → Desktop app: console.cloud.google.com/auth/clients
  6. Baixe o JSON → salve como client_secret.json

APIs para ativar:

📧 Gmail API
📅 Google Calendar API
💾 Google Drive API
📝 Google Docs API
📊 Google Sheets API
👤 People API (Contacts)
✅ Google Tasks API
📋 Google Forms API
🖼 Google Slides API
💬 Google Chat API
⚡ Apps Script API
🔧 Admin SDK API

2.3 Configurar o gog

# 1. Registrar as credenciais OAuth
gog auth credentials ~/Downloads/client_secret.json

# 2. Adicionar conta (abre navegador para autorizar)
gog auth add you@gmail.com --services gmail,calendar,drive,contacts,docs,sheets

# 3. Verificar
gog auth list

# 4. Conta padrão (adicionar ao .bashrc / .zshrc)
export GOG_ACCOUNT=you@gmail.com

2.4 Múltiplas contas

gog auth add work@empresa.com

# Usar conta específica em um comando
gog gmail search 'newer_than:1d' --account work@empresa.com

3📋 Referência rápida de comandos (terminal)

Para referência de admins e power users. Usuários finais não precisam memorizar isso — só falar com o agente em linguagem natural.
ServiçoComandoDescrição
Gmail gog gmail search 'newer_than:7d' --max 10 Últimos 7 dias (threads)
Gmail gog gmail messages search "in:inbox" --max 20 Mensagens individuais
Gmail gog gmail send --to a@b.com --subject "Hi" --body "Hello" Enviar e-mail
Gmail gog gmail send --to a@b.com --subject "Hi" --body-html "<p>...</p>" Enviar e-mail HTML
Calendar gog calendar events <calId> --from <iso> --to <iso> Listar eventos
Calendar gog calendar create <calId> --summary "Title" --from <iso> --to <iso> Criar evento
Drive gog drive search "query" --max 10 Buscar arquivos
Sheets gog sheets get <id> "Tab!A1:D10" --json Ler planilha
Sheets gog sheets update <id> "Tab!A1:B2" --values-json '[["A","B"]]' Escrever na planilha
Docs gog docs cat <docId> Ler documento
Contacts gog contacts list --max 20 Listar contatos
