# 🎓 OpenClaw por Bruno Okamoto — Kit Completo do Curso

**De zero a agente pessoal com OpenClaw.**

> Como eu construí uma assistente AI que substituiu 3-4 pessoas em 13 dias — e como você pode fazer o mesmo.

Por [Bruno Okamoto](https://www.youtube.com/@microsaas) 🍇

---

## 🚀 Como usar este repositório

1. **Assista a aula** na plataforma (Last Link)
2. **Encontre o material** na tabela abaixo — cada aula tem PDF, HTML e prompt
3. **Cole o prompt no seu OpenClaw** — o agente guia você pelo resto
4. **Use os use cases** pra aplicar no seu dia a dia

## 📂 Estrutura do Repositório

```
├── pdfs/                     # PDFs e HTMLs de todas as aulas
├── prds/                     # PRDs executáveis (joga no agente e ele faz)
├── prompts/                  # Prompts guiados por aula
├── templates/                # Context files prontos (SOUL, USER, AGENTS, etc.)
├── skills/                   # Skills de exemplo
│   ├── deep-research/        # Skill de pesquisa profunda
│   ├── seguranca/            # Skill de segurança
│   └── skills-by-profile.md  # Curadoria de skills por perfil
├── diagramas/                # Excalidraw (arquitetura visual)
├── configs/                  # Configurações de referência
├── docs/                     # Documentação master
├── qa/                       # Runbooks de diagnóstico
├── use-cases/                # Casos de uso reais com prompts
│   ├── business/             # Métricas SaaS, reports
│   ├── community/            # Análise de comunidade
│   ├── content/              # Content waterfall, ideias
│   ├── productivity/         # Daily briefing, social metrics
│   ├── research/             # Deep research
│   └── support/              # Análise de suporte
└── landing-pages/            # Landing pages de referência
```
