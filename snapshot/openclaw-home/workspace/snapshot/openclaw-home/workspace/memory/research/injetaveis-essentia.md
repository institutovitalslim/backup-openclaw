- Principais grupos mapeados e usos práticos:
  - Antioxidantes/metabólicos (ácido lipoico, NAC, glutationa, azul de metileno) → suporte mitocondrial, detox, sensibilidade à insulina, fadiga crônica.
  - Vitaminas hidrossolúveis/lipossolúveis (complexos B, vit C mega dose, vit E oleosa, CoQ10) → correção de deficiências, imunidade, neuropatias, performance cognitiva/cardiovascular.
  - Aminoácidos e blends (alanil glutamina, BCAA, HMB, L-arginina/citrulina, D-ribose, beta-alanina) → performance, massa magra, suporte hepático/renal, energia.
  - Neurotransmissores (5-HTP, triptofano, GABA, L-theanina, alfa-GPC, citicolina, PQQ+ATP) → modulação de sono, ansiedade, foco, SNC.
  - Modulares hormonais (anastrozol, dutasterida, finasterida, crisina, geranilgeraniol, HCG) → equilíbrio hormonal masculino/feminino, TRH, alopecia.
  - Peptídeos/fatores (EGF/IGF/TGF-B3, IGF+BFGF+VEGF+copper peptide) → protocolos estéticos/cicatrização/capilar.
  - Quelantes/detox (EDTA cálcico/sódico, DMPS, DMSO, citrato de sódio) → metais pesados, calcificações vasculares, suporte renal/hepático.
  - Minerais (cloretos, fosfato de potássio, cromo, ferro carboximaltose) → reposição eletrolítica, dislipidemias, anemia ferropriva.
  - Lipid-burn/estética corporal (fosfatidilcolina, desoxicolato, complexos Citrus sinensis, ioimbina, SAC cafeína) → gordura localizada, celulite, drenagem.
  - Dermatologia/estética facial (ácido tranexâmico, ácido kójico + TGP-2, DMAE, silício orgânico, biotina) → melasma, textura, queda capilar.
  - Protocolos EV/IM/ID pré-configurados pelo laboratório para imunidade, energia, cognição, desintoxicação hepática, emagrecimento, SNC, autoimunes, performance, saúde sexual, sono, cardiovascular etc.

Quando precisar sugerir uma formulação:
1. Buscar na planilha (`catalogo_injetaveis_lista.csv`) o ativo/combinação permitida para a via desejada.
2. Conferir se existe protocolo Essentia pronto (seção "protocolos" do PDF) para aproveitar proporções, diluição (geralmente 250 mL SF 0,9% EV) e frequência (1x/semana por 8–10 sessões).
3. Validar flags: exclusividade de via, veículo oleoso, necessidade de diluição lenta, controle especial.
