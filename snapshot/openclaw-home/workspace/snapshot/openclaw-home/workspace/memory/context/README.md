# Context

Arquivos de contexto guardam o que nao muda rapido.
Nao sao log diario e nao sao rascunhos.

Arquivos principais:
- decisions.md: regras permanentes e decisoes irreversiveis
- lessons.md: aprendizados e erros que nao devem se repetir
- people.md: equipe, parceiros e forma de interagir
- business-context.md: negocio, operacao e prioridades duraveis
- pending.md: itens esperando decisao ou retorno do Tiaro
