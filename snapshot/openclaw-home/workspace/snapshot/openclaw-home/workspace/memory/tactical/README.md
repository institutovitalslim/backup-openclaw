# Tactical Memory

Esta pasta e reservada para notas explicitamente taticas e temporarias.

## Quando usar

Use esta pasta para registros que:
- descrevem workaround temporario
- documentam incidentes operacionais pontuais
- tem valor curto de acompanhamento
- nao representam politica, decisao ou contexto duravel

## Regra De Retencao

Para que uma nota daqui possa ser removida automaticamente apos 30 dias, inclua o marcador abaixo nas primeiras linhas do arquivo:

Retention: tactical-30d

Sem esse marcador, o prune automatico nao deve apagar o arquivo.

## Nao guardar aqui

Nao use esta pasta para:
- decisions
- lessons
- people
- projects
- topic files
- feedback loops
- qualquer memoria que precise virar referencia de longo prazo
