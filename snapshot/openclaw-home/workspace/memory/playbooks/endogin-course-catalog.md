# endogin-course-catalog.md - Playbook

## Objetivo

Responder perguntas sobre modulos, produtos, trilhas e videos do Endogin sem reinventar ou chutar a estrutura.

## Ordem De Consulta

1. `memory/integrations/endogin.md`
2. `memory/projects/endogin-platform.md`
3. arquivos tematicos `memory/projects/endogin-*.md`
4. `memory/research/endogin-catalog-deep-index.md`
5. `memory/guide-endogin.md`
6. `snapshot/endogin/course-structure.md`
7. `snapshot/endogin/course-structure.json`

## Regras

- Se o usuario pedir lista de modulos e videos, usar `snapshot/endogin/course-structure.md`
- Se o usuario pedir trilhas por tema, usar os arquivos tematicos `memory/projects/endogin-*.md`
- Se o usuario pedir visao de plataforma, usar `memory/projects/endogin-platform.md`
- Se o usuario pedir absorcao, planejamento de estudo ou qual conteudo consumir, sugerir a partir dos grupos e trilhas ja mapeados
- Nao afirmar que o Endogin esta ausente da memoria enquanto estes arquivos existirem

## Resposta Esperada

- citar o grupo principal
- citar os produtos/modulos relevantes
- listar aulas/videos quando pedido
- apontar a fonte interna usada
