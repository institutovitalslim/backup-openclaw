# Playbook: Marketing Production Routine

## Fluxo padrao
1. Capturar briefing com memory/content/campaigns/canva-marketing-brief-template.md.
2. Ler voz da plataforma em memory/content/voice/.
3. Decidir entre reaproveitar design existente ou gerar do zero no Canva.
4. Produzir a peca no Canva.
5. Registrar links finais e aprendizados no projeto/campanha correspondente.

## Saidas esperadas
- link editavel no Canva
- link de visualizacao ou asset exportado
- copy principal
- CTA
- observacoes para iteracao

## Quando usar cada caminho
- Se ja existe um modelo bom: search-designs + get-design.
- Se precisa algo novo rapidamente: generate-design + create-design-from-candidate.
- Se precisa entrega final fora do Canva: get-export-formats + export-design.
