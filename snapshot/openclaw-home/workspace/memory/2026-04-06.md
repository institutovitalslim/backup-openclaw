#### Formato
- **1080 x 1350 px** (4:5)
- Fundo sólido: `#071A15` (verde-escuro quase preto)

#### Header fixo (todos os slides)
- Avatar circular ~44 px, topo esquerdo
- Nome: Inter/Open Sans Semibold, ~30 px, branco `#FFFFFF`
- @: Inter/Open Sans Regular, ~24 px, cinza `#A9B3AF`

#### Contador
- Topo direito
- Cápsula arredondada, fundo ~`#1A2A25`
- Texto branco ~24 px

#### Texto principal
- Sans-serif limpa (Inter ou Open Sans), Regular
- Tamanho: 56–68 px dependendo do slide
- Entrelinha: 1.2–1.3
- Cor: branco `#FFFFFF`
- Alinhado à esquerda
- Margem lateral: 70–90 px

#### Regra de posicionamento
- Texto curto → bloco (header + texto) desce verticalmente para centralizar visualmente
- Texto longo → bloco ancorado no topo normalmente

#### Ícone mute
- Canto inferior direito
- Tamanho ~24 px, cinza `#BFC6C2`
- Discreto

#### Estrutura da sequência (5 slides analisados)
- Slides 1 e 2: header + texto superior + foto inferior (integrada, arredondada, sem moldura)
- Slides 3–5: só header + texto + ícone mute (visual tipo "tweet real do X/Twitter")
