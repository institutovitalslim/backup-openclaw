# Endogin Platform

Ultima atualizacao: 2026-03-26

## Resumo

Plataforma autenticada da mentoria/area de membros Endogin, acessada por:

- URL de login: `https://membros.mentoriaendogin.com.br/auth/login?redirect=%2F`
- credencial armazenada em 1Password no item `Senha Acesso Endogin`
- acesso validado via 1Password CLI integrado ao app desktop do Windows

## Estrutura Geral

O catalogo autenticado esta organizado em vitrines e paginas internas no formato:

- home autenticada: `/area/vitrine`
- paginas de grupo/produto: `/area/produto/...`
- paginas de item/aula: `/area/produto/item/...`
