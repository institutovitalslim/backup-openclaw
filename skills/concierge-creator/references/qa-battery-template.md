# QA Battery Template

Create a scenario list before launch.

## Minimum scenarios
1. Direct price question
2. Insurance / reimbursement question if relevant
3. Hot lead ready to book
4. Cold lead asking generic information
5. Price objection
6. Prior-frustration objection
7. Logistics objection
8. Emotional/vulnerable lead
9. Return-after-silence lead
10. Fast path to booking

## Evaluation grid
For each scenario, assess:
- Sounds human?
- Sounds concierge?
- Answers clearly?
- Builds value?
- Avoids visible funnel feel?
- Handles price elegantly?
- Respects boundaries?
- Advances the goal?

## Output format
For each scenario, record:
- input message
- expected answer
- short evaluation
- score
- failure mode
- recommended fix
