# Starter Fields

Replace every `{{PLACEHOLDER}}` with business-specific information.

## Minimum fields to fill
- `{{CONCIERGE_NAME}}`
- `{{ROLE_DESCRIPTION}}`
- `{{PRIMARY_GOAL}}`
- `{{TARGET_ACTION}}`
- tone items
- price rules
- boundaries
- escalation rules
- recurring pains
- recurring objections
- channel openings
- hot lead signals
- QA verdict placeholders

Do not launch a concierge with placeholders still present.
