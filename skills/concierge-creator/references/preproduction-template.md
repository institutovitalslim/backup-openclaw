# Pre-Production Template

## Goal
Operate with supervision before full autonomy.

## Controls
- pause
- resume
- takeover
- release
- status

## What to monitor first
- tone drift
- repetition
- price handling
- trust signals
- missed opportunities to advance
- unsafe or out-of-scope statements

## Daily loop
- review key conversations
- record learnings
- identify recurring blockers

## Weekly loop
- review conversion patterns
- update prompt/brain if repeated evidence justifies it
