### Hooks Framework — Reels e Carrosseis
- File: `memory/content/hooks-reels-carrosseis.md`
- Committed: `070f0c6`
- Source: Tiaro (2026-04-06 17:32 UTC)

**8 tipos de hook catalogados:**
1. Quebra de Padrão — contraria o óbvio, cérebro trava
2. Dor Direta — pessoa se sente exposta, alta retenção
3. Curiosidade Aberta — loop mental, nunca entregar no hook
4. Autoridade Instantânea — prova + promessa rápida
5. Promessa Clara e Direta — simples > mirabolante
6. Número + Tempo + Resultado — gatilho lógico, números baixos (1,3,5)
7. Confissão / Vulnerabilidade — empatia, verdade estratégica
8. Alerta / Aviso — medo de perder, usar com responsabilidade

**Fórmula universal:** `Dor + Curiosidade + Promessa curta`

**Erros fatais:** começar se apresentando, contexto longo, frases genéricas, hook que entrega tudo imediatamente.

**Regra operacional:** toda criação de reel ou carrossel começa consultando `memory/content/hooks-reels-carrosseis.md` para escolher o tipo de hook correto antes de escrever a primeira linha.
